# Tools & Utilities

Reusable utilities and abstractions for paper processing.

## Subdirectories

### pdf_parsers/
Multiple PDF extraction backends with different strengths.

### markdown_generators/
Conversion and formatting utilities.

### chunking_strategies/
Various chunking algorithms with different optimization criteria.

### metadata_extractors/
Information extraction tools for paper metadata.

### evaluation_metrics/
Quality measurement and scoring systems.

## Architecture
These tools should be designed as composable, reusable modules that can be combined by the approaches in different ways.
