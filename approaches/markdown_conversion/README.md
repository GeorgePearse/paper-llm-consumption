# Markdown Conversion Approach

Transform PDFs into well-structured Markdown while preserving document hierarchy and formatting.

## Goals
- Preserve section structure (H1, H2, H3, etc.)
- Maintain formatting emphasis (bold, italics)
- Handle special content (math, code blocks)
- Keep page references for citations

## Key Challenges
- Detecting section hierarchies from PDF layout
- Handling multi-column layouts
- Converting LaTeX math to readable format
- Preserving figures and table references
- Dealing with page breaks
