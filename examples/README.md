# Examples

Sample conversions and output demonstrations.

## Subdirectories

### converted_papers/
Example papers processed through different approaches:
- Raw paper
- Markdown conversion output
- Hierarchically chunked version
- Semantically parsed representation
- Table extractions
- Figure descriptions

### metadata_samples/
Extracted metadata examples showing:
- Title, authors, publication info
- Keywords and topics
- Abstract
- Citation counts and references

### chunking_examples/
Demonstrations of different chunking strategies:
- Token-based chunks
- Semantic chunks
- Hierarchical chunks
- With and without overlap

## Usage
These examples serve as:
- Visual references for expected output
- Test cases for evaluation metrics
- Demonstration of approach capabilities
- Documentation of typical outputs
